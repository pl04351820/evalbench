-------- detailed description of this database ------

---- table: tbl_users ----

column `user_id` description: the primary key and unique identifier of the user
column `first_name` description: the first name of the user
column `last_name` description: the last name of the user
column `username` description: the username of the user for identification
column `password_hash` description: the hashed password of the user for security
column `social_media_links` description: JSON data storing links to the user's social media profiles
column `contact_info` description: JSON data storing contact information of the user (e.g., email, phone number)
column `avatar_url` description: URL pointing to the user's avatar or profile picture
column `bio` description: a brief biography or description of the user
column `website` description: the user's personal or professional website URL
column `created_at` description: timestamp indicating when the user account was created
column `updated_at` description: timestamp indicating when the user account was last updated
column `notification_preferences` description: settings for the user's notification preferences
column `role_id` description: the role or access level of the user in the system
column `last_login` description: timestamp indicating the user's last login date and time
column `active` description: flag indicating whether the user account is currently active or inactive

---- table: tbl_posts ----

column `post_id` description: the primary key and unique identifier of the post
column `user_id` description: foreign key referencing the user who created the post (linked to Users table)
column `title` description: the title of the post
column `content` description: the main content or body of the post
column `published_at` description: timestamp indicating when the post was published
column `is_published` description: flag indicating whether the post is published or not
column `views_count` description: the number of views/views count for the post
column `likes_count` description: the number of likes/likes count for the post
column `shares_count` description: the number of shares/shares count for the post
column `created_at` description: timestamp indicating when the post was created
column `updated_at` description: timestamp indicating when the post was last updated
column `slug` description: a URL-friendly version of the post title for SEO purposes
column `excerpt` description: a brief excerpt or summary of the post content
column `status_id` description: the status or status code of the post (e.g., draft, published)
column `category_id` description: foreign key referencing the category or topic of the post
column `dislikes_count` description: the number of dislikes/dislikes count for the post
column `comments_count` description: the number of comments/comments count for the post
column `is_featured` description: flag indicating whether the post is featured or not
column `author_name` description: the name of the post's author (if different from the user)
column `image_url` description: URL pointing to an image associated with the post
column `tags` description: JSON data storing tags or keywords associated with the post for categorization
column `reading_time` description: estimated reading time for the post
column `external_link` description: URL pointing to an external resource related to the post
column `metadata` description: JSON data storing additional metadata or information related to the post

---- table: tbl_comments ----

column `parent_comment_id` description: foreign key referencing the parent comment if this is a reply
column `likes_count` description: the number of likes/likes count for the comment
column `created_at` description: timestamp indicating when the comment was created
column `updated_at` description: timestamp indicating when the comment was last updated
column `is_approved` description: flag indicating whether the comment is approved or not
column `dislikes_count` description: the number of dislikes/dislikes count for the comment
column `report_count` description: the number of times the comment has been reported
column `is_deleted` description: flag indicating whether the comment is deleted or not
column `user_ip` description: the IP address of the user who posted the comment
column `user_agent` description: information about the user's browser or agent when posting the comment
column `is_flagged` description: flag indicating whether the comment is flagged for review or not
column `reply_count` description: the number of replies/reply count for the comment
column `mentions` description: JSON data storing mentions of other users in the comment
column `attachments` description: JSON data storing any attachments or files associated with the comment
